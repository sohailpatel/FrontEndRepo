Closing Tag

A new tag would not be able to be started without closing it.

For examples

<html>,<head>,<body>
<p>,<dt>,<dd>
<li>,<option>,<thead>
<th>,<tbody<,<tr>
<td>,<tfoot>,<colgroup>

Self Closing Tag

In HTML5 it is not strictly necessary to close certain HTML tags. The tags that aren’t required to have specific closing tags are called “self closing” tags.

For examples

<img>,<input>,<br>
<hr>,<meta>
